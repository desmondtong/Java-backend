# Login Registration App
Simple user login and registration app leveraging i18next and react-i18next that allows multi-lingual web app.

---
## Languages and Technologies Used

### Front-end
- React
- TypeScript
- CSS
- Material UI

### Back-end
- Java
- Spring Boot
- Spring Security
- JWT
- Database: PostgreSQL
- Driver: Node-progres (pg)

### Others
- React i18next
